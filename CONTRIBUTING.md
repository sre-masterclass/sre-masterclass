# Contributing

## Naming Conventions

- **Services**: `[service-name]-api` (e.g., `ecommerce-api`)
- **Components**: `[ComponentName].jsx` (e.g., `ProductList.jsx`)
- **CSS Files**: `[ComponentName].css` (e.g., `ProductList.css`)
- **Scenarios**: `[scenario-name].yml` (e.g., `cascading-failure.yml`)
